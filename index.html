  • ¿Qué es PUG?

    • Pug es un motor de plantillas para HTML, el cual es un intérprete y un compilador de códigos, el que nos facilita la escritura de códigos HTML para poder crear proyectos en el menor tiempo posible.


  • ¿Como funciona PUG?

    • Pug nos permite observar una salida HTML con una sintaxis simple donde no se tienen etiquetas, además usando tabulaciones para indicar que elementos están dentro de otros.
